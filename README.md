## Install
Clone this project

```bash
> git clone https://github.com/Stalker-moment/Chemicfest8_API/
> cd Chemicfest8_API
```

Install the dependencies:

```bash
> npm install
```

### Usage
1. run the Code

 Hard Mode

```bash
> npm start
```

 Nodemon 

```bash
> nodemon app.js
```

2. force stop the code
```bash
> ctrl + c
```
